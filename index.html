Css font sets

Responsive typography sets, works with any HTML.
All css font-properties, line-heights, colors, e.t.c. are configurable with Sass.

Why this package? Making html-formatted text look great is hard work.
Use this package to make html from RTEs and other sources work well on different screen sizes.
